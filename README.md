## 📌 Overview

**Batchitt** is a real-time chat extension for developers that lets you communicate, share code, and collaborate instantly right inside your code editor.

Chat 1:1 or in groups without leaving your coding environment 🚀

---

## 🔐 Trusted & Secure: Authentication via batchitt.com only

---

## ✨ Features

- 💬 One-to-one and group chat features within the code editor
- ⚡ Lightweight and fast integration inside code editor
- 🔗 Seamless collaboration within your workspace
- 🔐 Secure authentication using auth key
- 👨‍💻 Developer-friendly UI

---

## 🔐 Authentication Process

To use Batchitt, follow these steps:

1. Visit 👉 https://batchitt.com  
2. Login to your account  
3. Copy your **Auth Key** from dashboard  
4. Open you code editor
5. Paste the Auth Key inside the Batchitt extension  
6. You will be logged in successfully ✅  

---

## 🔐 Security & Trust

Batchitt uses secure authentication via auth key.
No sensitive data is stored locally.

Official Website: https://batchitt.com
Source Code: https://github.com/krishnaarga/batchitt

---

## 🛠️ Installation

### From VS Code / Cursor/ Antigravity Marketplace
1. Open you code editor
2. Go to Extensions (`Ctrl + Shift + X`)
3. Search for **Batchitt**
4. Click **Install**
